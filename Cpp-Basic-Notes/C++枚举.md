# C++枚举

### 什么是枚举

枚举是用于定义命名常量的派生数据类型。主要是提高代码的可读性和可维护性。用法：
```
enum {
    A, B, C // A=0, B=1, C=2
}
int main() {
    A;
    B;
    C;
    return 0;
}
```

### 注意
1. 枚举不是一个命名空间，但是可以通过完全限定名访问。
2. 枚举不是基本数据类型
3. 枚举类待补充

### 细节
枚举默认4字节。可以修改枚举的大小，如：
```
enum A : char{} // 1byte
```
