# 303. 区域和检索 - 数组不可变

前缀和的简单题

### 初次提交
```
class NumArray {
private:
    vector<int> m_prefixSum;
public: 
    NumArray(vector<int>& nums) {
        int total = 0;
        for (auto num : nums) {
            total += num;
            m_prefixSum.push_back(total);
        }
    }
    
    int sumRange(int left, int right) {
        int leftData = 0;
        if (left > 0) {
            leftData = m_prefixSum[left - 1];
        } else {
            leftData = 0;
        }
        return m_prefixSum[right] - leftData;
    }
};
```
